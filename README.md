# Vue Contact Form

A **simple, interactive contact form** built with Vue 3 for portfolio purposes.  
This project demonstrates **reactivity, form validation, computed properties, and async workflows** in Vue.

---

## 🔹 Features

- **Reactive inputs** using `v-model`
- **Validation** for empty fields and email format
- **Computed property** disables submit button when form is incomplete
- **Async submission simulation** with loading state
- **User feedback**: success message after submission
- Clean and responsive styling

---

## 🛠 Tech Stack

- Vue 3 (with `<script setup>`)
- Vite (build tool)
- HTML / CSS
- JavaScript (ES6+)
